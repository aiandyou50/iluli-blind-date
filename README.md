# Iluli Blind Date (이루리 소개팅)

English / 한국어 README — both included below.

---

## English

Iluli Blind Date is a Light & Fast Next.js 15 (App Router) dating app demo built for Cloudflare Pages with Edge functions.

Key features
- Simple, mobile-first UI built with Tailwind CSS.
- Authentication with NextAuth.js (Google provider).
- D1 (SQLite) via Prisma for database; Cloudflare R2 for image storage.
- Serverless / Edge-compatible API routes.
- Lightweight profile/photo/matching flows for demo use.

Tech stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Cloudflare Pages + Edge Runtime
- Cloudflare D1 (Prisma)
- Cloudflare R2 (image storage)
- NextAuth.js (v5) with Prisma adapter

## 한국어

이루리 소개팅은 Cloudflare Pages와 Edge runtime을 대상으로 만든 가벼운 Next.js 15 (App Router) 기반 데모 소개팅 애플리케이션입니다.

주요 기능
- 모바일 우선 설계의 Tailwind CSS UI
- NextAuth.js (Google) 기반 인증
- Prisma로 연결된 Cloudflare D1 (SQLite), 이미지 저장은 Cloudflare R2
- Edge 환경에서 동작하는 API 라우트
- 프로필, 사진 업로드, 매칭 흐름을 포함한 데모용 기능

기술 스택
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Cloudflare Pages + Edge Runtime
- Cloudflare D1 (Prisma)
- Cloudflare R2
- NextAuth.js (v5) + Prisma adapter
